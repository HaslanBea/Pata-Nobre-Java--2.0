# Pata-Nobre-Java--2.0# 🐾 Pata Nobre 2.0 (Java)

## 📌 Descrição do Projeto

O **Pata Nobre 2.0** é um sistema desktop desenvolvido em Java com o objetivo de auxiliar na gestão de um pet shop. O sistema permite o cadastro de clientes, pets e o agendamento de serviços, facilitando o controle e organização das informações.

---

## 🎯 Objetivo

Desenvolver uma aplicação simples e funcional para gerenciamento de pet shop, aplicando conceitos de programação orientada a objetos, interface gráfica e usabilidade.

---

## 🖥️ Funcionalidades

* 📋 Cadastro de clientes
* 🐾 Cadastro de pets
* 📅 Agendamento de serviços
* 🔄 Navegação entre telas
* 💬 Mensagens de confirmação de ações

---

## 🧩 Estrutura do Sistema

O sistema é composto pelas seguintes telas:

* **Tela Principal:** menu de navegação
* **Tela Cliente:** cadastro de clientes
* **Tela Pet:** cadastro de pets
* **Tela Agendamento:** agendamento de serviços

---

## 🛠️ Tecnologias Utilizadas

* Java
* Java Swing (interface gráfica)
* VS Code (ambiente de desenvolvimento)

---

## ▶️ Como Executar o Projeto

1. Abra o terminal na pasta do projeto
2. Compile os arquivos:

```
javac *.java
```

3. Execute o sistema:

```
java Main
```

---

## ⚠️ Observações

Devido a limitações de hardware, o projeto foi desenvolvido utilizando VS Code em vez de NetBeans, mantendo todas as funcionalidades propostas.

---

## ♿ Acessibilidade

O sistema foi desenvolvido considerando boas práticas de acessibilidade:

* Interface simples e intuitiva
* Botões com tamanho adequado
* Uso de textos claros e legíveis
* Navegação direta entre telas
